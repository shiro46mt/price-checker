# お買い得比較 (Price Checker)

スーパーなどで商品の容量と価格を入力し、グラム単価を比較するシンプルなWebアプリです。

大容量品と通常品で、どちらがお得かを買い物中にすぐ確認できます。

## Features

- 容量(g)と価格(円)を入力すると自動で円/gを計算
- 複数商品を同時比較
- 最安の商品に★マークを表示
- 最後の入力行に入力すると自動で行追加
- すべてクリア機能
- スマホのテンキー入力対応
